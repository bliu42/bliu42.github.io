Necessary condition for seperability

LOO: Local orthogonal observables.
PPT: Positive partial transposition

PPT criterion: a necessary condition for seperability.
For $\rho_{AB}$, if $\rho^{T_B}{AB}$ is still a density matrix (i.e. it is hermitian and has non-negative spectrum), then $\rho{AB}$ is seperable.

（非常有意思的一点是，因为是只对$B$做转置，所以取了转置符号$T$的一半，$\Gamma$来表示这样的操作，即将$\rho^{T_B}_{AB}$记作$\rho^\Gamma _{AB}$.）

PPT condition is equivalent to demanding the positivity of $(I_A\otimes T_B)\rho_{AB}$, where $T_B$ is the transposition map acting on B. $T_B$ is a positive (P) map, but not a completely positive (CP) map.

The map $\Theta$ is completely positive if and only if $I \otimes \Theta$ is positive for identity map $I$ on any finite-dimensional system.

Any P but not CP map $\Lambda_B$ can form a nontrival necessary condition for seperation:
$$(I_{A} \otimes \Lambda_{B}) \rho_{A B} \geq 0\tag{1}$$

Then we can provide a necessary and sufficient condition: $\rho_{AB}$ is seperable if and only if $(1)$ is satisfied for all P but not CP maps. However to find all the P but not CP maps $\Lambda_{B}$ is very hard for systems with dimension bigger than 6.

Entanglement witness

An entanglement witness is an oberservable that has:

  • (i) non-negative expectation values in all separable states.
  • (ii) at least one negative eigenvalue.
