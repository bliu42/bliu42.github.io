开放系统：密度矩阵和主方程

发表于 2021-08-31 更新于 2021-09-03 分类于 科研笔记 阅读次数：

A note on open system quantum mechanics.

1. Densitry Matrix

A mixed state is an ensemble of pure states.
The density matrix is defined as:
$$\rho = \sum_i p_i |\psi_i\rangle \langle \psi_i| \tag{1.1}$$

  • $|\psi_i\rangle$ not necessarily orthogonal.
  • $\sum_i p_i = 1, 0 \leq p_i \leq 1$, the possibility.
  • There are infinite ways to decompose the mixed state $\rho$ into a ensemble of pure states.

For pure states, $\rho^2 = \rho$ while for mixed states, $\rho^2 \neq \rho$.

The purity of the state is defined as $\mathcal P = \text{Tr}[\rho^2]$. For pure state $\mathcal P = 1$, while for mixed states $\mathcal P < 1$.

Some properties of mixed state density matrix:

  • Trace is 1: $\text{Tr}[\rho] = 1$.
  • Hermiticity: $\rho^{\dagger} = \rho$.
  • Positivity: For arbitary $|\psi\rangle$, $\langle \psi |\rho | \psi\rangle \geq 0$.

1.1 Partial Trace

Reduced density matrix is the density matrix for the subsytem. For a bipartite system $AB$ in state $\rho$, the state of $A$ can be get by tracing out the $B$ space:
$$\rho_A = \text{Tr}_B(\rho) = \sum_b \langle b | \rho | b \rangle \tag{1.2}$$
Here $|b\rangle$ is a set of orthogonal basis in the Hilbert space of B, $\mathcal H_B$. The $\rho_A$ is independent of the choice of $|b\rangle$.

1.2 The Schmidt Decomposition

Theorem: For any vector $v$ in Hilbert space $\mathcal H_1 \otimes \mathcal H_2$, there exist orthonormal sets on each space ${u_i^1},{u_i^2}$ with $1 \leq i \leq m\leq min{D_1,D_2}$, such that $v$ can be written as $v=\sum_{i=1}^m a_i u_i^1\otimes u_i^2$ with $a_i$ non-negative.

The number $m$ of vectors needed for the decomposition is called the Schmidt rank and the ${a_i}$ are Schimdt coefficients. If the Schmidt rank of a vector is one, the associate state is separable. And ${a_i^2}$ are the eigenvalues of the reduced density matrix obtained by taking the partial trace over the other system.

2. Lindblad Master Equation

Assuming the system is Markovian, we can expect the density matrix evolution to be linear:
$$\frac{d}{dt} \rho_s = \mathcal G \rho_s \tag{1.3}$$
To make the Markovian assumption appropriate, we only look at a time scale $T_s$ with $\Delta t_{env} \ll \Delta t_{sys} \ll T_s$.

For closed system, linear evolution gives the Liouville-von Neumann eqution:
$$\frac{d}{dt} \rho_s(t) = -i[H, \rho_s(t)]\tag{1.4}$$
For open system, the evolution can be written as an operation $\mathcal L_{\Delta t}$ :
$$\frac d {dt} \rho_s = lim_{\Delta t \to 0}\frac{\rho_s(t+\Delta t)-\rho_s(t)}{\Delta t}=\frac{\mathcal L_{\Delta t}[\rho_s(t)]-\rho_s(t)}{\Delta t}\tag{1.5}$$
And the operation $\mathcal L_{\Delta t}$ can be written in Kraus’ form:
$$\mathcal L_{\Delta t} (\rho_s) = \sum_{\alpha}E_\alpha \rho_s E_{\alpha}^{\dagger} = \rho_s +\Delta t \frac{d\rho _s}{dt}|_t +O(\Delta t^2)\tag{1.6}$$

with

$$\sum_{\alpha}E_\alpha^\dagger E_\alpha = I\tag{1.7}$$

The closed system evolution $(1.4)$ corresponds to one Kraus operator $E_0 = I - iH\Delta t$. For open system, the first Kraus operator can be $E_0 = I-iK\Delta t +O(\Delta t^2)$. Construct two Hermi operator using $K$:
$$H=\hbar (K+K^\dagger)/2, A = i(K-K^\dagger)/2 \tag{1.8}$$
So $K=H/\hbar-iA$, and (with nature unit $\hbar = 1$) $E_0 = I-(iH+A)\Delta t+O(\Delta t^2)$. For other Kraus operator with $\alpha \geq 1$, let $$E_\alpha = \sqrt{\Delta t} L_\alpha \tag{1.9}$$
And using $(1.7)$ we can get $A=\frac 1 2 \sum_{\alpha \geq 1}L_\alpha^\dagger L_\alpha$,
$$E_0 = I+(-iH-\frac 1 2 \sum_{\alpha \geq 1}L_\alpha^\dagger L_\alpha)\Delta t \tag{1.9}$$
Here ${L_\alpha}, 1\leq \alpha \leq N^2-1$ are referred to as “Lindblad operators” or “jump operators”. The evolution function $(1.5)$ then is:
$$\frac d {dt} \rho_s = -i[H,\rho_s]-{A,\rho_s}+\sum_{\alpha \geq 1}L_\alpha \rho_s L_\alpha^\dagger\tag{1.10}$$
This is the famous Lindblad master equation.

  • With only the first term, $(1.10)$ describes the closed system, where the evolution is unitary.
  • With only the first two terms it describes a non-Hermitian Halmitonian, which causes the decay of the amplitude and is not physical.
  • With all three terms it describes the evolution of an open system. The third term corresponds to the so-called “jump operators”.
  • For a paticular system and bath coupling system, the choice of the jump operators is not unique.
