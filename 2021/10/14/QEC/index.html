Stochastic Majorana noise models

Naively: $\gamma_i$ with same $p$ for each step. However, such a model will on average make the state spend an equal amount of time in each parity: even and odd. In physical systems, there is a finite gap between two different parity, making the naive model unrealistic.

Introduce the concept of MZM islands.

4 types of errors:

  • Quasiparticle event: application of a single MAjorana operator: $|\psi \rangle_M \to \gamma_{j,a}|\psi \rangle_M$.
  • Pair-wise dephasing event: application of a pair of Majorana operators belonging to the same island: $|\psi \rangle_M \to \gamma_{j,a}\gamma_{j,b}|\psi \rangle_M$.
  • Correlated event: application of Majorana operators from multiple islands involved in the same measurement.
  • Measurement bit-flip: flipping of the classical bit storing the outcome of a 2k-MZM parity measurement: e.g. $\vec m \to \vec m \oplus (0,1,0,\dots ,0)$.

For a Majorana circuit, the possible noise can be described as follow:

Majorana circuit noise (MC): in a given step for k-island measurement (k = 0 for an idle island), implement the following sequence:

    1. For each island that begins the time step with odd parity, apply a quasiparticle event with probability $p_{odd}^{(k)}$.
    1. For the set of islands involved in the same k-island mea- surement, do the following:
    • (a) For each island in the set, apply one single-island noise event: either a quasiparticle event with $p_{qp}^{(k)}$ or a pair-wise dephasing event with $p_{pair}^{(k)}$.
    • (b) Apply a correlated event to the set: either an odd correlated event with probability $p_{cor,odd}^{(k)}$ or an even one with $p_{cor,even}^{(k)}$. For $k\leq 1$, do nothing.
    1. Apply the measurement projector perfectly, then flip the classical bit storing the measurement outcome with probability $p_{mst}^{(k)}$. For an idle island, do nothing.

MC is defined by the parameter set ${p_{odd}^{(k)},p_{qp}^{(k)},p_{pair}^{(k)},p_{cor,odd}^{(k)},p_{cor,even}^{(k)},p_{mst}^{(k)}}$.

Physical System

Take the j-th tetron (a 4-MZM island) and its Hamiltonian can be written as:

$$H = E_C(\hat n_s - n_g)^2 + \sum_k E_k\hat n_{\Delta,k} + \sum_{a \neq b} \delta E_{ab}i\gamma_{j,a}\gamma_{j,b}$$

  • The first term is the charging energy. $n_s$ is the number of fermions on the island when the island is not grounded (i.e. Coulomb-blockaded).
  • The second term is the BCS Hamiltonian written in terms of the quasiparticle number operator $\hat n_{\Delta,k}$. $\hat n_{\Delta,k}$ counts the above-gap quasiparticles.
  • The last term is the MZM hybridization energy. $\delta E_{ab}$ is set by the overlap of $\gamma_{j,a}$ and $\gamma_{j,b}$. According to the topological protection, $\delta E_{a b} \propto e^{-L_{a b} / \xi}$.
  • The simple model does not include the overlap of MZMs from different islands.

The eigen states of the Hamiltonian can be written as:

$$\left|n_{s},\left{n_{\Delta, k}\right}{\Delta}\right\rangle\left|i \gamma{j, 1} \gamma_{j, 2}, i \gamma_{j, 3} \gamma_{j, 4}\right\rangle$$

The first part only contains the non-topological part, and the second part denotes the (almost) degenerate MZM space. How we pair up the MZMs (1,2 and 3,4) is chosen randomly.

We can set $n_g = 0$. The gs is:

$$|g\rangle |\psi_i\rangle_M = |0,{0}_\Delta \rangle |\psi_i\rangle_M$$

where $i =0, 1$ denotes the odd and even parity.
